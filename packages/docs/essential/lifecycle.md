# 生命周期


微信小程序生命周期顺序
1. App onLaunch
2. App onShow
3. Behavior created  <- 按数组顺序加载
4. Component created
5. Behavior attached
6. Component attached
7. Page onLoad
8. Component pageLifetime show
9. Page onShow
10. Behavior ready
11. Component ready
12. Page onReady
13. Page onHide
14. Component pageLifetime hide
15. Page onUnload
16. Behavior detached
17. Component detached
